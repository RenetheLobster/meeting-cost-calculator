# Meeting Cost Calculator

Data viz tool calculating the true cost of project management meetings with stakeholders. UK & USA salary data.

## What It Does

Calculates the real cost of bringing stakeholders together for project meetings based on:
- Role salaries (UK & USA averages)
- Meeting duration
- Number of attendees
- Overhead costs (room, travel, tech)

## Why It Matters

Most organisations don't realise how expensive meetings actually are. This tool exposes the hidden costs.

## Features

- 💰 Real-time cost counter during meetings
- 📊 UK & USA salary benchmarks
- 👥 Stakeholder role templates
- 📈 Visual cost breakdowns
- 💾 Export reports

## Tech Stack

- Python (data processing)
- Streamlit/Plotly (visualisation)
- JSON (salary data)

## Quick Start

```bash
pip install -r requirements.txt
python app.py
```

## Data Sources

- UK: ONS, Glassdoor, Indeed
- USA: BLS, Glassdoor, PayScale
